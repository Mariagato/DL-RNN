# DL-RNN
Proyecto #2 - semana 4

## Entorno Python

Este notebook fue creado con Python 3.11 y kernel `rnn_env`.

1. Instalar Python 3.11 desde https://www.python.org/downloads/
2. Crear el entorno virtual:

```powershell
python -m venv .venv
```

3. Activarlo:

```powershell
.\.venv\Scripts\Activate.ps1
```

4. Instalar dependencias:

```powershell
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

5. Registrar el kernel para Jupyter:

```powershell
python -m ipykernel install --user --name rnn_env --display-name "rnn_env"
```
